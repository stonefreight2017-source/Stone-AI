How Voice Chat Works

🎤 Speech-to-Text (Input)

User taps the mic button. Browser's SpeechRecognition API captures audio and converts to text. The text appears in the chat input and auto-sends.

Works in: Chrome, Edge, Safari 14.1+

🔈 Text-to-Speech (Output)

When Bestie responds, SpeechSynthesis API reads the response aloud. User hears the Bestie's words. A "Speaking..." indicator shows it's active.

Works in: All modern browsers
